# Roadmap

## Current Phase: Multiplayer Hardening

- Improve online room UX and reliability.
- Keep the server authoritative for all online moves.
- Maintain local and bot modes while online mode matures.
- Preserve backend-ready storage boundaries for future persistence.

## Next Build Tasks

- Polish create/join/reconnect flows.
- Make waiting, opponent turn, spectator, and disconnect states obvious.
- Add copy-room-code feedback.
- Expand tests around online UI edge cases.
- Keep documentation current as decisions change.

## Later Backlog

- Database-backed room persistence.
- Public lobbies and private room settings.
- Player names and lightweight profiles.
- Timers and draw/move-limit rules.
- Rematch flow and tournament/session stats.
- Hosted deployment.

## Out Of Scope For Now

- Accounts and authentication.
- Ranked matchmaking.
- Chat.
- Cross-device persistent profiles.
